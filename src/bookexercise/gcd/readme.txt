计算最大公约数

input.txt
第一行为正整数n表示n个测试用例
后续n行每行包含两个正整数，用于指定需要求最大公约数的两个数

Euclidean法，又叫辗转相除法
r = a % b
如果r为0则b为最大公约数
否则令a = b, b = r再次计算
问题：如何证明方法正确性
效率：高

更相减损法
1.如果两个数都是偶数。两个数都除以2.直到不全是偶数，保存除的若干个2的倍数
2.以较大的数减较小的数，把差与较小的数比较，如果较小的数与差相等，则停止。
否则以差与减数中的较大者作为被减数，较小着作为减数继续运算
3.将第一步中除掉的倍数乘以第二步中的差。得到最大公约数
问题：如何证明方法正确性
效率：？
